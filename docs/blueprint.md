# **App Name**: IBRBTV Replay

## Core Features:

- Live Stream Display: Display the current live stream from the specified YouTube channel on the main screen.
- Video Catalog: Create a catalog of previously broadcast videos available for replay.
- Video Selection Menu: Implement a menu for users to select and play previous videos.
- Video Ingestion: Provide a simple tool to input/update the catalog of previous video URLs. Use AI to summarise videos.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to create a professional and trustworthy atmosphere.
- Background color: Very light blue (#E8EAF6) to complement the primary color and provide a clean backdrop.
- Accent color: Soft purple (#8E24AA) to draw attention to important elements and create a unique visual identity.
- Headline font: 'Poppins', sans-serif, to keep the aesthetic fresh.
- Body font: 'PT Sans', sans-serif, provide a professional and friendly impression.
- Use simple and modern icons for navigation and video controls.
- Prioritize a clean and intuitive layout, ensuring easy navigation and clear video presentation.
